Color Schema

Color should always be used in meaningful and intentional ways in order to create patterns and visual cues.

Brand

Use it only as a special accent, in order to highlight some secondary elements or where the Stratio logo is required to display in its colored version.

Space

The Space grey is available for secondary or tool buttons.

Action

Use this colors for actionable buttons or links.
The blue ones are for primary actions such as CTAs.

Status

Status colors are available to inform the user about system status, such as error messages, health checks or notifications.
The red ones allows you to alert the user about critical processes, and can be used as well in deleting buttons.

Neutral

Neutral colors can be used for layout purposes. You can use it in backgrounds, line spacers, headers, and this kind of layout elements in order to bring the interface to the rhythm it needs.

Misc

A variety of colors to be used in graphs like table headers, charts or other tipe of non-primary-interactive informative elements.
