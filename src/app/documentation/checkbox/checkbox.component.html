Checkbox

Checkboxes are used when there is a list of options and the user may select multiple options, including all or none.

Design & Behavior

Checkboxes have five states. Default state, Checked, Indeterminate, Disabled and Disabled checked.
The can be grouped inline or one under another.

Usage and Best Practices

Always use clear and concise labels for Checkboxes. Be explicit about the action that will follow if the Checkbox is selected. Labels appear to the right of Checkboxes.

Users should be able to select the checkbox by clicking on the box directly or by clicking on its label.

The default view of a set of Checkboxes is having no option selected.
