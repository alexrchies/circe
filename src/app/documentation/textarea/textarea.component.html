Textarea

Textarea inputs are used for freeform data entry. They anticipate the user to input more than one sentence or more than one line of text.

Design & Behavior

We show a text area in forms where users need to enter sometimes lengthy data, such as information that may run over to two or more lines. They can be resized by the user and the rest of the content should adapt to this. Text areas can have multiple states depending on the situation. Read only, disabled, on focus state, show error messages, etc.

Usage and Best Practices

For accessibility, text labels are required for all text area items. Use clear, concise, consistent labels to guide the user. If the subject is complicated, find words to make it simpler. Each label is left-justified and appears above the text area it describes.
